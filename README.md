# Bug-Tracking System.-
The Bug Tracking System is a software application used to identify, manage, and track bugs or errors in a project. It helps developers and testing teams record issues, assign tasks, and monitor the progress of fixing bugs. This system improves communication between team members and helps maintain software quality.

The project is designed to make bug management simple and organized. Users can report bugs, update their status, assign them to developers, and track completed fixes in one place.
